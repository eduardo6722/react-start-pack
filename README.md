## Description

[Challenge](https://github.com/t10d/fullstack-assessment) T10 Full-stack Engineer Challenge.

## Installation

```bash
$ yarn | npm install
```

## Running the app

### With Docker

```bash
$ docker build -t frontend .
```

```bash
$ docker run -it --network=host frontend
```

### Local

```bash
# development
$ yarn start | npm run start
```
